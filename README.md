# invoice-ideoDigital

Invoice control app which has the following capabilities:
<ol>
<li>create clients</li>
<li>create customers</li>
<li>create invoices</li>
<li>edit invoices</li>
<li>present customers</li>
<li>present invoices</li>
</ol>

Tech stack:
<ol>
  <li>Backend
      <ol>
        <li>Python - flask</li>
        <li>Prisma - ORM</li>
      </ol>
    </li>
  <li>Frontend
    <ol>
    <li>React</li>
    </ol>
  </li>
  <li>Database
    <ol>
    <li>MySQL</li>
    </ol>
  </li>
</ol>
